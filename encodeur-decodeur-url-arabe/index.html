Rendez vos liens internet contenant des mots arabes à nouveau lisibles (Décodage) ou préparez-les pour les intégrer dans du code (Encodage).

Tester le lien

Pourquoi les liens internet arabes deviennent-ils illisibles?

Si vous avez déjà copié l'adresse d'un article Wikipédia en arabe ou le lien d'une vidéo YouTube avec un titre en arabe pour l'envoyer sur WhatsApp ou Facebook, vous avez remarqué qu'un lien court comme domaine.com/مرحبا se transforme en une ligne extrêmement longue ressemblant à domaine.com/%D9%85%D8%B1%D8%AD%D8%A8%D8%A7.

Le système d'Encodage d'URL (Percent-encoding)

L'architecture originelle d'Internet (créée dans les années 1990) a défini que les URLs (les liens web) ne pouvaient contenir qu'un jeu très restreint de caractères occidentaux: les lettres de l'alphabet latin (A-Z, a-z), les chiffres (0-9) et quelques symboles spécifiques (-, _, ., ~).

Les caractères de l'alphabet arabe, cyrillique, ou asiatique n'étant pas autorisés, les navigateurs internet (Chrome, Safari) appliquent un processus appelé "URL Encoding". Chaque lettre arabe est convertie en une séquence de codes hexadécimaux précédés du symbole pourcentage (%). Par exemple, la lettre arabe م devient systématiquement %D9%85.

Comment utiliser notre outil?

Cet outil gère les deux sens de conversion, instantanément et localement dans votre navigateur:

1. Décoder (Rendre l'URL lisible)

Sélectionnez "Décoder". Collez votre lien illisible rempli de pourcentages. L'outil va traduire les codes informatiques et révéler les véritables mots arabes. Vous pouvez ensuite copier ce lien "propre" pour le partager visuellement dans un article de blog, sur X (Twitter) ou dans un document de recherche afin qu'il soit esthétique.

2. Encoder (Sécuriser l'URL)

Si vous êtes développeur web, référenceur (SEO) ou programmeur (PHP, Python), et que vous devez insérer un lien arabe dans le code source d'une application ou d'une base de données, vous ne devez jamais laisser les lettres arabes brutes. Sélectionnez "Encoder". L'outil sécurisera le texte arabe en code pourcentage (Percent-encoding) pour éviter tout bug serveur (erreurs 404).

Astuce de test

Dès que vous décodez ou encodez un lien valide commençant par http:// ou https://, un bouton bleu "Tester le lien" apparaîtra. Vous pourrez cliquer dessus pour vérifier instantanément si le lien généré mène toujours à la bonne page web.
